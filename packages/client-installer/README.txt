RPrint Client Installer
=======================

This installer creates shortcuts to access RPrint from your computer.

WINDOWS INSTALLATION:
---------------------
1. Double-click "install-windows.bat"
2. Click "Yes" if prompted for permissions
3. Look for RPrint icon on your Desktop and Start Menu
4. Click the icon to open RPrint

MAC INSTALLATION:
-----------------
1. Open Terminal
2. Navigate to this folder
3. Run: chmod +x install-mac.sh
4. Run: ./install-mac.sh
5. Find RPrint in your Applications folder

WHAT THIS DOES:
---------------
- Creates shortcuts to open https://growingsoft.net in your browser
- No separate application to maintain or update
- Uses your default web browser
- Secure HTTPS connection
- All features of the web version

UNINSTALL:
----------
Windows: Delete the shortcuts from Desktop and Start Menu
Mac: Drag RPrint.app to Trash from Applications folder
